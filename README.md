# Xyner 模块增强功能集(老板会员可直接使用)

本模块是一个基于 Xposed 的增强工具，兼容主流社交与短视频应用（如微信、抖音、快手、小红书等），专为提升使用体验与功能自由度而设计。

### 📱 微信（WeChat）增强功能

#### 🔧 密友(守护家庭和谐)

#### 🎨 界面设置

* **全局透明头像**：优先级大于全局圆形头像

#### ⏰ 聊天时间样式增强
* **使用系统自带的时间 View 显示**

### 📹 抖音（DouYin）增强功能

#### 🚀 基本功能
* **去除启动和闪屏广告**

#### 🎨 界面相关
* **隐藏上下黑边**
* **去除更新弹窗**
* **去除青少年弹窗**

#### 🎬 视频筛选
* **过滤总开关**
* **中断开关**：筛选后数量不足以显示,就会跳过此次筛选
* **去广告开关**
* **去直播开关**
* **去推荐卡片开关**
* **去推荐商家开关**
* **去小游戏开关**
* **去长视频开关**
* **长视频时间界定**：默认＞2分钟视频(单位秒)
* **去低赞视频开关**：实际赞<设置赞,将会去除
* **去图集开关**
* **过滤字符串**：文案,音乐标题,作者名,等等出现字符串就会去除该视频

#### 💾 下载相关
* **去除下载限制**
* **无水印下载图集或者视频**

#### ▶️ 自动播放
* **自动滑动到下一个视频**

#### 💬 自动回复
* **开启自动回复**
* **回复模式**：请在Xyner菜单设置内容
* **回复预设内容**
* **关键词回复**
* **随机回复**：从预设池随机挑选一条
* **延迟回复**：默认2秒,太快会造成还没显示就已经回复了
* **AI提示词**：例如:你是一个聊天高手，请用简短自然的语气回复对方。
* **回复后是否Tip提醒**
* **全局自动回复**：打开后,所有人的消息都会回复
* **AI回复历史记录条数**：默认20轮 越大,每次使用的token越多,但是回复的内容越精确

#### 💭 自动评论
* **开启自动评论**：启用后，系统会自动在符合条件的视频下发表评论
* **评论模式**
* **预设评论内容**
* **随机评论内容**：随机评论内容
* **AI评论提示词**：用于AI生成评论的上下文提示，例如"你是一位幽默的短视频评论助手,请简短的内容回复视频"
* **评论延迟时间**：评论前的延迟时间（秒），避免过快操作被检测
* **评论后是否提示**：评论成功后是否显示提示信息

#### 🎯 判断条件
* **我关注的用户**
* **关注我的用户**
* **互相关注用户**
* **含关键词的视频**
* **关键词列表**：填写关键词列表，用于匹配标题或描述
* **点赞数条件**：仅当视频点赞数<=设置数量时才评论
* **评论数条件**：仅当视频评论数<=设置数量时才评论
* **每日评论上限**：限制每天最多自动评论数量，防止异常频率 小于1为不限制
* **关键词回复**
* **视频抢首评**：没有评论时才执行
* **无视任何条件直接执行**：所有条件都失效

#### ✍️ 评论相关
* **评论自动添加后缀**
* **后缀内容**
* **评论输入框 # 开头自动转语音发送**：在评论输入框中以 # 开头输入文字，发送时自动将文字转为语音评论发送

#### 🔊 评论长按语音功能
* **下载语音**：长按语音评论可直接下载语音文件到本地
* **复读语音**：长按语音评论一键复读，自动下载并以语音评论形式重新发送
* **收藏语音**：长按语音评论收藏到本地收藏夹，支持自定义备注（无评论内容时默认使用作者昵称）
* **语音收藏夹**：管理已收藏的语音，支持从收藏夹中直接发送语音评论
* **评论转语音发送**：长按任意文字评论，将评论内容通过 TTS 语音合成后以语音评论形式发送（自动过滤 `[表情]` 标记，仅合成纯文字内容）

#### 🎮 播放控制
* **禁用双击点赞**
* **不允许系统跳过已隐藏视频**
* **不允许中途插入视频**
* **拦截长按倍数启用场景**：拦截后,无法使用抖音自带的长按倍数
* **自动倍速播放**：切换视频自动按照设置的倍速播放
* **倍速数值**

#### 👁️ 界面隐藏
* **隐藏总开关**：打开后以下隐藏选项生效，取值范围说明：
  * **-2**：完全隐藏且不占据布局空间，相邻 View 会自动贴合。
  * **-1**：完全隐藏但仍占据布局空间，相邻 View 不会移动。
  * **0 ~ 255**：设置透明度，0 为全透明，255 为完全不透明。
* **统一透明度开关**
* **统一透明度**
* **点赞按钮透明度**
* **评论按钮透明度**
* **头像透明度**
* **收藏按钮透明度**
* **分享按钮透明度**
* **音乐按钮透明度**
* **作者名透明度**
* **视频文案透明度**
* **全屏按钮透明度**
* **合集按钮透明度**
* **底部栏透明度**
* **顶部栏透明度**
* **底部评论区透明度**
* **进度条透明度**
* **长按事件**
* **双击事件**
* **打开评论时,不滑动视频**
* **顶部搜索栏**
* **切换视频时,自动执行隐藏View**
* **显示View时,自动打开评论列表**
* **暂停播放时,显示View**
* **共创透明度**
* **沉浸式全屏**
* **好友分享的顶部提示透明度**
* **底部私信回复透明度**
* **评论性别**：没有显示性别的用户不执行

#### 📨 消息管理
* **消息防撤回**：模块如果掉了,可能会出现已经被阻止的 消息,再次被撤回
* **自己撤回的消息**
* **对方撤回的消息**
* **撤回的背景色**

### 🪐 Soul（灵魂）增强功能

#### 🛡️ 纯净社交
* **去广告与弹窗**：全面去除应用内广告、更新提醒弹窗及青少年模式弹窗。
* **社交降噪**：禁止"弹出挚友邀请"，禁用系统强制的"智能打招呼"。
* **隐身专家**：上线自动进入隐身状态，悄无声息浏览星球。

#### 🎭 伪造本地资源 (老板会员功能)
* **数据自定义**：本地修改并显示你的**关注数、被关注数、获赞数**及**新增获赞统计**。
* **足迹模拟**：自定义显示"看过我"的总量及**最新访客**数据。
* **伪造总开关**：一键启用或关闭所有数据修改项，快速切换真实/伪造状态。

#### 🔓 权限解锁
* **本地 VIP 特权**：开启本地 VIP 标识及相关特权功能。
* **社交无限制**：解除聊天频率限制，解除内容不可分享的锁定。
* **校园吧权限**：突破限制，自由访问各学校校园吧。
* **自动签到**：每日自动完成任务，保持活跃等级。

#### 👁️ 信息透视
* **账号背景查阅**：实时显示对方的**注册天数**、**昵称**以及**封禁剩余时间**。
* **Token 获取**：支持一键提取当前账号 Token，方便高级调试。

#### 🔍 高阶筛选
* **精准匹配**：支持根据**性别、在线状态、注册天数区间**（设置上下限）进行筛选。
* **用户去重**：自动过滤 **SSR 用户、VIP 用户、已关注或粉丝**，提升交友效率。

#### 💬 聊天黑科技
* **消息防撤回**：拦截并保留对方撤回的消息，且**不受两分钟限制**。
* **防撤回视觉定制**：支持自定义防撤回消息的**背景颜色**，一眼识别撤回内容。

## ⚙️ 使用说明

1. 安装 [LSPosed](https://github.com/LSPosed/LSPosed) 框架
2. 安装本模块并在 LSPosed 中启用
3. 在作用域中勾选需要增强的目标应用
4. 重启对应应用或设备以生效
5. 可通过模块配置界面开启/关闭各项功能

## 📌 注意事项

* 所有功能皆可自由开关配置
* 运行环境要求：LSPosed（或兼容的 Xposed 环境）
* 系统版本要求：Android 8.1+（建议更高版本以获得更稳定体验）
* 针对特定应用或版本的功能可能暂不可用或表现差异，遇到问题可暂时关闭对应功能，再反馈具体版本信息与复现步骤
* 与其他 Hook/模块发生冲突时，建议逐一排查或减少重叠功能
* 网络相关能力仅用于调试与问题定位，请在法律与平台政策允许范围内使用
* 请关闭框架的 Xposed API 调用保护
* 本模块不会上传或收集任何用户数据
