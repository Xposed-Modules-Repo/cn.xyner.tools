# Xyner 模块增强功能集(老板会员可直接使用)

本模块是一个基于 Xposed 的增强工具，兼容主流社交与短视频应用（如微信、抖音、快手、小红书等），专为提升使用体验与功能自由度而设计。

## 支持应用

* 微信（WeChat）
* 抖音（DouYin）
* 快手（KuaiShou）
* 小红书（XiaoHongShu）
* 多邻国（DuoLingo）
* 以及更多正在适配的 App 和功能（持续更新中）

## ✨ 功能总览（理论支持所有最新版本）

### 📱 微信（WeChat）增强功能

#### 🔧 基础设置
* **禁用热更新**：默认开启，如果不开启可能会出现有些功能无法使用

#### 🎨 界面设置
* **全局圆形头像**：将头像显示为圆形
* **全局透明头像**：优先级大于全局圆形头像

#### ⏰ 聊天时间样式增强
* **使用系统自带的时间 View 显示**
* **调整原始时间 View 的上下间距**
* **设置时间显示的偏移位置**：仅在时间未显示于聊天中部时生效
* **隐藏系统自带的时间 View**：关闭使用系统自带的时间View后才生效
* **将时间显示在聊天框的中间位置**
* **设置自定义时间 View 的背景颜色**
* **设置自定义时间 View 的文字颜色**
* **深色模式下的自定义时间背景颜色**：未设置时沿用上方颜色
* **深色模式下的自定义时间文字颜色**：未设置时沿用上方颜色
* **设置自定义时间背景的圆角大小**
* **设置自定义时间文字的大小**
* **创建时间显示组件（显示在消息下方）**：在聊天消息下方新增一个时间显示组件，用于展示消息的时间信息
* **自定义时间格式**：设置时间显示的格式，不填写时默认采用 [HH:mm:ss]

#### 🌐 朋友圈相关
* **去除朋友圈广告**
* **朋友圈消息列表**：主页增加朋友圈消息列表直达菜单
* **清除未读**
* **主页加号添加模块菜单**
* **朋友圈列表点击进入详情页**

#### 🛡️ 聊天防撤回
##### 背景色模式
* **开启防撤回背景色改变**
* **背景色颜色**：支持透明度

##### 灰色提示模式
* **开启防撤回灰色小字提醒**
* **消息防撤回自定义灰色提示**

#### 👥 群聊相关
* **历史聊天记录**
* **资料页复制wxid**
* **强制相机扫码**
* **监控好友发朋友圈**：点开好友资料页可以设置是否监控(如果是密友状态只会显示微信团队)
* **后台自动刷新朋友圈间隔**：默认10秒一次(修改后重启生效)
* **朋友圈自动刷新**
* **显示群主管理员**
* **群主显示头衔**
* **管理员显示头衔**
* **群主头衔显示颜色**
* **管理员头衔显示颜色**

#### 🎯 其他功能
* **隐藏顶部和底部Tab**：隐藏微信顶部和底部的Tab栏

---

### 📹 抖音（DouYin）增强功能

#### 🚀 基本功能
* **去除启动和闪屏广告**

#### 🎨 界面相关
* **隐藏上下黑边**
* **去除更新弹窗**
* **去除青少年弹窗**

#### 🎬 视频筛选
* **过滤总开关**
* **中断开关**：筛选后数量不足以显示,就会跳过此次筛选
* **去广告开关**
* **去直播开关**
* **去推荐卡片开关**
* **去推荐商家开关**
* **去小游戏开关**
* **去长视频开关**
* **长视频时间界定**：默认＞2分钟视频(单位秒)
* **去低赞视频开关**：实际赞<设置赞,将会去除
* **去图集开关**
* **过滤字符串**：文案,音乐标题,作者名,等等出现字符串就会去除该视频

#### 💾 下载相关
* **去除下载限制**
* **无水印下载图集或者视频**

#### ▶️ 自动播放
* **自动滑动到下一个视频**

#### 💬 自动回复
* **开启自动回复**
* **回复模式**：请在Xyner菜单设置内容
* **回复预设内容**
* **关键词回复**
* **随机回复**：从预设池随机挑选一条
* **延迟回复**：默认2秒,太快会造成还没显示就已经回复了
* **AI提示词**：例如:你是一个聊天高手，请用简短自然的语气回复对方。
* **回复后是否Tip提醒**
* **全局自动回复**：打开后,所有人的消息都会回复
* **AI回复历史记录条数**：默认20轮 越大,每次使用的token越多,但是回复的内容越精确

#### 💭 自动评论
* **开启自动评论**：启用后，系统会自动在符合条件的视频下发表评论
* **评论模式**
* **预设评论内容**
* **随机评论内容**：随机评论内容
* **AI评论提示词**：用于AI生成评论的上下文提示，例如"你是一位幽默的短视频评论助手,请简短的内容回复视频"
* **评论延迟时间**：评论前的延迟时间（秒），避免过快操作被检测
* **评论后是否提示**：评论成功后是否显示提示信息

#### 🎯 判断条件
* **我关注的用户**
* **关注我的用户**
* **互相关注用户**
* **含关键词的视频**
* **关键词列表**：填写关键词列表，用于匹配标题或描述
* **点赞数条件**：仅当视频点赞数<=设置数量时才评论
* **评论数条件**：仅当视频评论数<=设置数量时才评论
* **每日评论上限**：限制每天最多自动评论数量，防止异常频率 小于1为不限制
* **关键词回复**
* **视频抢首评**：没有评论时才执行
* **无视任何条件直接执行**：所有条件都失效

#### ✍️ 评论相关
* **评论自动添加后缀**
* **后缀内容**

#### 🎮 播放控制
* **禁用双击点赞**
* **不允许系统跳过已隐藏视频**
* **不允许中途插入视频**
* **拦截长按倍数启用场景**：拦截后,无法使用抖音自带的长按倍数
* **自动倍速播放**：切换视频自动按照设置的倍速播放
* **倍速数值**

#### 👁️ 界面隐藏
* **隐藏总开关**：打开后以下隐藏选项生效，取值范围说明：
  * **-2**：完全隐藏且不占据布局空间，相邻 View 会自动贴合。
  * **-1**：完全隐藏但仍占据布局空间，相邻 View 不会移动。
  * **0 ~ 255**：设置透明度，0 为全透明，255 为完全不透明。
* **统一透明度开关**
* **统一透明度**
* **点赞按钮透明度**
* **评论按钮透明度**
* **头像透明度**
* **收藏按钮透明度**
* **分享按钮透明度**
* **音乐按钮透明度**
* **作者名透明度**
* **视频文案透明度**
* **全屏按钮透明度**
* **合集按钮透明度**
* **底部栏透明度**
* **顶部栏透明度**
* **底部评论区透明度**
* **进度条透明度**
* **长按事件**
* **双击事件**
* **打开评论时,不滑动视频**
* **顶部搜索栏**
* **切换视频时,自动执行隐藏View**
* **显示View时,自动打开评论列表**
* **暂停播放时,显示View**
* **共创透明度**
* **沉浸式全屏**
* **好友分享的顶部提示透明度**
* **底部私信回复透明度**
* **评论性别**：没有显示性别的用户不执行

#### 📨 消息管理
* **消息防撤回**：模块如果掉了,可能会出现已经被阻止的 消息,再次被撤回
* **自己撤回的消息**
* **对方撤回的消息**
* **撤回的背景色**

## ⚙️ 使用说明

1. 安装 [LSPosed](https://github.com/LSPosed/LSPosed) 框架
2. 安装本模块并在 LSPosed 中启用
3. 在作用域中勾选需要增强的目标应用
4. 重启对应应用或设备以生效
5. 可通过模块配置界面开启/关闭各项功能
